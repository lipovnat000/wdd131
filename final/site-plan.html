Overview

Purpose

This site is going to be a simple picture puzzle game. The picture will be split into a grid and shuffled and the user will have to swap 2 pieces at a time until the image is complete.

Audience

The audience will be anyone looking for a quick to play puzzle game.

Dynamic elements

If possible, I'd like to split an image automatically into different squares, but if I can't figure it out I'll just manually save the individual grid squares separately using some sort of naming scheme. Regardless of how dynamic the game is set up, the javascript will have to a) allow the user to swap two pieces by clicking two squares in sequence and b) check if the puzzle is complete. If I have time I'd like to create a hard mode too, using a 7x7 square instead of a 4x4 square in the default difficulty, and that would be chosen with a button on the main screen.

The script will also have to save the user's choices on the home screen and determine the puzzle based on those choices, as well as lock the play button until the user has chosen a difficulty and image.
