Overview

Purpose

The purpose of this site is to expand the existing content of my YouTube channel. I want to include behind the scenes images/videos, embedded videos from my YouTube channel, and a potential news page. If I were ever to have an audience large enough to sell merchandise, I'd want to include that too, although not for this assignment.

Audience

Viewers of my YouTube channel interested in expanded content.

Dynamic elements

Automatically sorting videos based on criteria set by the user as well as pulling information like title, thumbnail, and viewcount from the YouTube API.
